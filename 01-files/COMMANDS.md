# Day 1 - Linux Files and Folders

## Commands practiced

- pwd = show current location
- ls = list files
- cd = change directory
- mkdir = create folder
- touch = create empty file
- echo = print text
- cat = show file content
- file = show file type
- rm = remove file

## What I learned

Linux is mainly about moving through folders, creating files, reading files, and understanding where I am.

Today I also learned that accidental words can become files if I run the wrong command.
