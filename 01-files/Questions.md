# 01-files - Questions and Answers

## 1. What does pwd do?

pwd shows my current folder location.

## 2. What does ls -la show?

ls -la shows all files, including hidden files, with details like permissions, owner, size, and date.

## 3. What is the difference between > and >>?

> overwrites the file.
>> adds new text to the end of the file.

## 4. What does cat do?

cat prints the content of a file in the terminal.

## 5. Why is grep useful?

grep helps me search for specific text inside a file. For example, I can search for ERROR inside a log file.

## 6. What does find do?

find searches for files and folders. For example, find . -name "*.log" finds log files from the current folder.

## 7. What did I practice in this section?

I practiced creating files, reading files, copying files, renaming files, searching inside files, and finding files.
